A full-stack dashboard built with MERN, Redux, Ant Design, Tailwind, Node-Cron, and Socket.IO.

🚀 Features
Authentication & Authorization
Secure JWT-based login/signup
Role-based access (Admin/User)
CRUD Operations
Create, Read, Update, Delete resources
Fully responsive UI with Tailwind & Ant Design
Real-Time Updates
Instant notifications & live updates via Socket.IO
Task Scheduling
Automated background tasks using Node-Cron
State Management
Redux Toolkit for global state & async API calls
Additional Utilities
Form validation, toasts, error handling, and secure password hashing
🛠️ Tech Stack
Layer	Technology / Library
Frontend	React.js, Redux Toolkit, Tailwind CSS, Ant Design
Backend	Node.js, Express.js, Socket.IO, Node-Cron
Database	MongoDB, Mongoose
Auth	JWT, bcrypt
Tools	axios, react-router-dom, dotenv, cors, nodemon
